Describe the difference between addEventListener('click', ...) and .onclick

`addEventListener('click', ...)` and `.onclick` are two different ways to add a click event handler to an HTML element.

The `.onclick` property is a DOM Level 0 property that sets a function to be called when the element is clicked. It can only be set to a single function and will overwrite any previously set click event handler.

The `addEventListener('click', ...)` method is a DOM Level 2 method that allows you to attach multiple event handlers to an element. It takes two arguments: the type of the event you want to listen for (in this case, 'click'), and the function to be called when the event is triggered. This method also allows you to remove the event listener later on if needed.
